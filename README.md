## About

ShoreSquad is a youth-focused beach cleanup community platform that mobilizes young people to clean beaches using weather tracking, interactive maps, and social features to make eco-action fun and connected.

## Project Overview

- **Target Audience**: Young eco-warriors (Gen Z & Millennials)
- **Mission**: Make beach cleanups social, easy, and impactful
- **Features**: Interactive maps, weather tracking, squad building, impact tracking

## Getting Started

### Prerequisites
- A modern web browser
- [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) VS Code extension (recommended)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd dev-c240
```

2. Open in VS Code:
```bash
code .
```

3. Launch with Live Server:
   - Right-click on `index.html`
   - Select "Open with Live Server"
   - Or click "Go Live" in the status bar

The site will open at `http://localhost:5500`

## UX Design Principles

1. **Mobile-First**: Designed for smartphones first
2. **High Contrast**: Optimized for outdoor visibility
3. **Clear CTAs**: Action-oriented buttons and messaging
4. **Progressive Disclosure**: Information revealed as needed
5. **Feedback**: Visual feedback for all interactions
6. **Consistency**: Unified design language throughout
